# node-bcrypt

`npm init -y`

`npm i -S fastify`

`npm i -S bcrypt`  

`npm run dev`

Dockerfile

`docker build . -t nodebcrypt`

`docker run -dp 1400:4000 nodebcrypt`

`curl http://localhost:4000/bcrypt/:password`

E.g.
`curl http://localhost:4000/bcrypt/thana`
